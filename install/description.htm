Code Chef Extension to speed up CFML coding.

Code Chef cooks up CFML code for you in no time.
Just select Code Chef from dynamic Views drop down and click Show View to get started.
Once Code Chef view is launched just type the action/keyword in Code Chef view (ex. write file, create pdf, loop, or get current directory), select appropriate option from suggestions and hit enter.
Appropriate code will be inserted in any file already open in editor.

Enjoy!!!
